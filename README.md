# AI Job Market Analytics Using Python

## Project Overview

This project analyzes a real-world AI Job Market dataset containing 10,345 job postings using Python.

The objective is to identify salary trends, skill demand, hiring patterns, experience-level impact, and other key factors influencing the AI job market.

Using Exploratory Data Analysis (EDA), the project transforms raw data into actionable business insights through visualizations and statistical analysis.

---

## Dataset Information

* Records: 10,345
* Features: 19
* Countries: USA, Canada, UK, Germany, India, Australia, Singapore
* Job Roles:

  * AI Engineer
  * Machine Learning Engineer
  * Data Scientist
  * Data Engineer
  * Data Analyst
  * Business Analyst

---

## Business Questions Addressed

1. Which AI roles offer the highest salaries?
2. Which countries pay the highest salaries?
3. How does experience level impact compensation?
4. Does education level affect salary?
5. Which skills increase salary the most?
6. Do remote jobs pay more than onsite jobs?
7. How does hiring urgency influence salary?
8. What are the overall hiring trends in the AI industry?

## Key Findings

### Highest Paying Roles

* AI Engineer → $139,945
* Machine Learning Engineer → $139,705

### Experience Impact

* Senior → $138,289
* Mid → $113,592
* Entry → $89,096

### Highest Salary-Boosting Skills

* Deep Learning → +$15,223
* Machine Learning → +$14,605
* Cloud → +$9,653

### Hiring Trends

* High urgency roles pay nearly $10K more.
* Remote, Hybrid, and Onsite jobs have very similar salary ranges.

## Business Recommendations

* Focus on Deep Learning and Machine Learning skills for higher compensation.
* Gain practical experience to maximize salary growth.
* Monitor high-urgency openings for better opportunities.
* Prioritize skills and experience over academic qualifications alone.
